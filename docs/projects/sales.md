# Intelligent Sales Forecasting

## Project Overview
A 15TB end-to-end data project using Microsoft Fabric.

!!! success "Key Technical Win"
    Processed 15 terabytes of enterprise data using a three-tier Medallion Architecture.

!!! note "Technical Stack"
    * **Data:** PySpark, Fabric Data Factory
    * **AI:** MLflow, Auto-ML
    * **Reporting:** Power BI Direct Lake

## The Solution
I moved the logic from static fine-tuning to an **Agentic RAG workflow**, allowing the AI to query the live Gold-layer Lakehouse for real-time demand forecasting.
